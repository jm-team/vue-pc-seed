# vue-seed

> Vue2 PC版项目脚手架

## 简单几步构建基础项目

### 安装依赖包  
```bash
$ yarn install
# or 
$ yarn
```

### 生成Dll文件  
```bash
$ yarn run build:dll
```

### 开发环境  
```bash
$ yarn run dev
```

### 生产环境  
```bash
$ yarn run build
# then
$ yarn run start
```

### 代码校验 ESlint  
```bash
$ yarn run lint
```

### 代码修复  
```bash
$ yarn run lint:fix
```

## 开发配置
* 代码构建配置提取到`config/index.js`
* 开发环境配置提取到`config/address.config.js`
